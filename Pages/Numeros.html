un (1) → /œ̃/ (se pronuncia como un sonido nasal entre "an" y "un").
- **deux** (2) → /dø/ (similar a "dö", con un sonido de "e" cerrada).
- **trois** (3) → /tʁwa/ (suena como "truá").
- **quatre** (4) → /katʁ/ (se pronuncia "katr", sin vocal al final).
- **cinq** (5) → /sɛ̃k/ (suena como "sank" con un toque nasal).
- **six** (6) → /sis/ (suena como "sis", aunque a veces es más como "si" dependiendo del contexto).
- **sept** (7) → /sɛt/ (se pronuncia "set").
- **huit** (8) → /ɥit/ (suena como "üit", con el sonido "ü").
- **neuf** (9) → /nœf/ (similar a "nöf", con el sonido de "e" cerrada).
- **dix** (10) → /dis/ (se pronuncia "dis").
- **onze** (11) → /ɔ̃z/ (suena como "onz", nasal).
- **douze** (12) → /duz/ (similar a "dus", con una "u" francesa).
- **treize** (13) → /tʁɛz/ (suena como "tres", pero más abierta).
- **quatorze** (14) → /katɔʁz/ (se pronuncia "katorz", con la "o" francesa).
- **quinze** (15) → /kɛ̃z/ (suena como "kanz", nasal).
- **seize** (16) → /sɛz/ (similar a "ses", pero más suave).
- **dix-sept** (17) → /dis sɛt/ (se pronuncia "dis set").
- **dix-huit** (18) → /dis ɥit/ (suena como "dis üit", con el sonido "ü").
- **dix-neuf** (19) → /dis nœf/ (se pronuncia "dis nöf").
- **vingt** (20) → /vɛ̃/ (suena como "van", nasal).
- **30** - **trente** → /tʁɑ̃t/ (se pronuncia "trant", con un sonido nasal).
- **40** - **quarante** → /kaʁɑ̃t/ (se pronuncia "karant", con un toque nasal).
- **50** - **cinquante** → /sɛ̃kɑ̃t/ (se pronuncia "sankant", nasal en ambas sílabas).
- **60** - **soixante** → /swasɑ̃t/ (se pronuncia "suasant", nasal).
- **70** - **soixante-dix** → /swasɑ̃t dis/ (se pronuncia "suasant dis").
- **80** - **quatre-vingts** → /katʁə vɛ̃/ (se pronuncia "katre van", nasal).
- **90** - **quatre-vingt-dix** → /katʁə vɛ̃ dis/ (se pronuncia "katre van dis").
- **100** - **cent** → /sɑ̃/ (se pronuncia "san", nasal).
